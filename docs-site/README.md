# 五行人格卡项目文档站

这个目录提供一个零依赖静态文档站，用来展示类似文档平台的左侧目录 + 右侧正文布局。

首页已经串起项目定位、H5 体验、架构证据、八小时工作流、从零学习手册、PPT 资产包、数据中台和压测材料入口，适合作为作品集说明站。

## 本地查看

直接用浏览器打开：

```text
docs-site/index.html
docs-site/showcase.html
```

## GitHub Pages

如果仓库启用 GitHub Pages，并选择从 `main` 分支根目录发布，可以访问：

```text
https://Oliver1205.github.io/wuxing-persona-card/docs-site/
```

GitHub 仓库首页 README 本身不能自定义真实侧边栏，这个目录用于承载更专业的项目说明站。
