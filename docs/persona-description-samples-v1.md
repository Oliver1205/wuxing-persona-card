# 五行人格描述样稿 v1

> 目的：先随机挑 10 个主副点睛组合验证文案质感，再决定后续扩展到 64 种、81 种或其他数量。以下不是最终命格库。

## 生成原则

- 每个结果都按“主元素 + 辅元素 + 点睛元素”描述。
- 主元素讲人格底色，辅元素讲协作方式，点睛元素讲让这个人更立体的亮点。
- 先解释年份、月份、可选日期/时段和 5 题答题为什么会推到这个主次结构。
- 年份使用干支纪年和纳音取象，月份使用节令/月令取象；这是人格产品的轻量模型，不冒充完整八字排盘。
- 相生写成推动、滋养、点亮；相克写成边界、分寸、校准。
- 不说缺陷、冲突、命弱、过旺、补救、改运。

## 新格式样稿

输入假设：2005 年 12 月，未填日期和时段；5 题里 3 题偏火、2 题偏土。  
判定：2005 年取干支为乙酉，乙木主生发、酉金主清肃，纳音泉中水作年份主气；12 月近大雪冬至，子水当令、寒土收束；5 题火向最多、土向补强，故形成主火、次土。主火代表活力斗志，次土带来稳重执行。火去探索，土来落地，是敢冲也能守基础的命盘。

## 审稿重点

你可以重点看四件事：

1. 名字是否有记忆点。
2. 文案是不是“像人格分析”，不是像玄学断语。
3. 点睛元素有没有让结果更立体。
4. 正向表达是否自然，是否过度夸张。
