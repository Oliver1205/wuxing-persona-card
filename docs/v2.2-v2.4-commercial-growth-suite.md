# v2.2-v2.4 商业增长与生产体验增强

日期：2026-06-11

分支：`codex/v2.2-v2.4-commercial-growth-suite`

## 1. 阶段目标

在 v2.1 已完成事件归因和后台增长漏斗的基础上，本阶段把项目推进到更接近商业级运营的状态：

- v2.2：把原始事件沉淀为日聚合指标，降低后台趋势查询成本。
- v2.3：补齐上线前的 smoke、备份、恢复、回滚和 Nginx 限流安全基线。
- v2.4：强化移动端首屏、结果身份感和分享海报体验。

本阶段仍然不做登录、付费、AI 深度解读、朋友匹配、社区和复杂 BI。

## 2. v2.2 增长聚合

新增表：

- `site_daily_metric`
- `short_link_daily_metric`

关键规则：

- `visit_event` 仍是唯一明细事实源。
- 聚合表只保存已闭合日期，禁止聚合今天或未来日期。
- 后台日趋势优先读取历史日聚合，今天或缺失日期回退实时事件。
- `metricSource` 用于说明趋势来源：`daily_metric`、`live_event`、`mixed`。
- 管理端提供手动刷新聚合入口：`POST /api/admin/analytics/aggregate`。

## 3. v2.3 生产硬化

新增脚本：

- `scripts/production-smoke-test.sh`
- `scripts/backup-mysql.sh`
- `scripts/restore-mysql.sh`
- `scripts/deploy-rollback.sh`
- `scripts/mobile-e2e.sh`

Nginx 增强：

- `/api/**`、`/api/events`、`/s/**` 分区限流。
- 增加基础安全响应头。
- 限制请求体大小。
- 设置代理超时，避免慢请求拖垮后端。

## 4. v2.4 前端体验

增强点：

- 首页强化 90 秒、5 题、身份名、五行比例和短链分享的首屏预期。
- 结果页增加人格身份摘要、一句话人格感和更适合截图传播的结构。
- 分享模块突出复制、系统分享和保存图三个入口。
- Canvas 分享图升级为更完整的海报布局，并对长短链自动换行。

## 5. 验收标准

- 后端测试通过。
- 前端构建通过。
- Docker Compose config 可解析。
- 质量脚本覆盖新增脚本语法检查和禁词扫描。
- 管理后台可触发聚合并展示趋势来源。
- 生产 smoke、备份、恢复、回滚脚本具备明确输入和失败提示。

## 6. 后续保留

- Flyway 或 Liquibase 正式迁移治理。
- Playwright 依赖安装和 CI 自动运行。
- HTTPS 证书落地、HSTS、线上备份恢复演练。
- 告警和日志平台。
